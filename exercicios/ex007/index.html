Principais Formatações

Negrito / Destacado

Nesta Frase utilizamos o B Para deixar Negrito não (semântico)

Nesta Frase Usamos o STRONG Para deixar negrito fazendo seu uso (semântico)

Italico / Ênfase

Nesta Frase Usamos o Italico para deixar italico fazendo o uso não (semântico)

Nesta frase usamos o italico Em para deixar italico fazendo o uso (Semântico)

Texto Marcado

Podemos fazer o uso do Marcador de texto Mark para fazer a marcação

E no outro parágrafo, temos outro texto marcado no final.

Podemos mudar a cor do marcador atraves do uso das CSS.

Texto Grande / Pequeno

Aqui estamos fazendo o uso do big que vai cair em desuso porém a tag Small não ira sair, apenas aceita que doi menos

Texto Deletado

Podemos usar a tag del para texto não tão importante porém que ainda podem ser lidos.

Texto inserido

Aqui usamos a tag U para sublinhas para usa não (semântico).

Aqui usamos a tag ins para deixar sublinando com uso semântico, onde se usa tbm para prestar bastante atenção no texto

Texto Sobrescrito

Para texto Sobre escrito usamos o sup, Exemplo: X20 = 0

Texto Subscrito

Agora para texto Subscrito usamos o sub e o texto fica assim exemplo: H 2O
